# Animal Rescue UI/UX Design Project

## Overview
A complete end-to-end UI/UX design for an animal rescue platform focused on improving animal adoption, reporting, and care management workflows in Palestine.

## My Role
UI/UX Designer

## Design Process
- Conducted user flow planning and information architecture
- Created low-fidelity wireframes to define structure and navigation
- Designed high-fidelity UI screens with a consistent design system
- Built a complete design system (spacing, components,ect..)

## Key Screens
- Sign Up / User Registration flow
- Adoption Requests Management system
- Organization Profile page
- User Profile page
- Animal listing and adoption workflows

## Focus Areas
- Usability and user experience improvement
- Accessibility and clean interface design
- Simplified adoption and reporting process
- Consistent visual hierarchy and responsive layout structure
